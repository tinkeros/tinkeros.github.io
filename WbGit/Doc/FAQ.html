Frequently Asked Questions


-] Why TinkerOS?
  It can run on more bare metal 64-bit PC's than TempleOS.
-] Why not backport feature X to TempleOS?
  There is not official TempleOS continuation to contribute to. TinkerOS is 
  backwards compatible with TempleOS to the maximum extent possible.
-] What TempleOS applications are not compatible with TinkerOS?
  Applications which attempt to hi-jack the graphics system and use the old VGA 
  registers.
-] Why doesn't resolution X work?
  You can only use VBE supported graphics modes, use the VBE tool on the USB 
  boot image to find out what will work with your monitor/system.
